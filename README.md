# 2023OSSL
2023-01 OSSL Team Project


-mini project 주제
CRUD를 이용한 주차관리 시스템

-mini project에 대한 소개
일상생활에서 사용하는 주차관리 시스템을 CRUD형식을 사용하여 효율적으로 사용하여 요금계신, 자신의 자동차 위치 찾기 등 유저가 사용하기 위한 환경 구축하는 것을 목표로 한다. 또한 사용자가 자신의 차를 찾기 쉽게 그림으로 자신의 차를 볼 수 있게 만든다는 것에서 의의가 있다.
